SpeechFeatures.jl

SpeechFeatures.jl is a Julia package for extracting acoustic features for speech technologies.

Authors

  • Lucas Ondel (website

Installation

The package can be installed with the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run:

pkg> add SpeechFeatures
